💻 Example Code (Python)

def plusOne(digits): for i in range(len(digits)-1,-1,-1): if digits[i] < 9: digits[i] += 1 # done return digits digits[i] = 0 # carry return [1] + digits # all 9s

⚠️ Edge Cases

[0] → [1] single zero
[9] → [1,0] single nine
[1,0,0] → [1,0,1] no carry
[9,9,9] → [1,0,0,0] full carry
[2,4,9,9] → [2,5,0,0] partial

🎯 When to Use

  • Increment a large integer as digit array.
  • Carry propagation right → left.
  • Avoid integer overflow with arrays.

📊 Time/Space Complexity

⏱️ Time: O(N)
Single right-to-left pass
💾 Space: O(1)
In-place (O(N) all-9s)

💡 Key Insight

  • Only 9s cause carry.
  • digit < 9 → increment & return immediately.
  • Loop ends → all 9s → prepend 1.
